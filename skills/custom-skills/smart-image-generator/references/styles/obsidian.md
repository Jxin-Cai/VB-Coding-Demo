# 风格：Obsidian（默认）

## 概览

**obsidian** 风格体现了知识工作、笔记和智力探索的美学。受 Obsidian.md 和 Roam Research 等工具启发，这种风格使用简洁的手绘线条、最少的颜色和深思熟虑的构图，创造出感觉像属于个人知识库的图像。

## 视觉特征

### 线条质感
- **手绘精准**：线条感觉人工制作但有意图
- **一致重量**：简洁、中等重量的笔触
- **平滑曲线**：几何但不完全机械
- **黑色或深灰**：很少为线条使用颜色（#1A1A1A 到 #333333）

### 配色方案
- **背景**：浅色、中性（白色、米白、浅灰）
  - 主要：#FFFFFF 或 #F5F5F5
  - 从不繁忙或有纹理
- **线条**：深灰到黑色（#1A1A1A, #2D2D2D）
- **强调色**：最少、策略性使用单一强调色
  - 通常：紫色（#7C3AED）、蓝色（#3B82F6）或青色（#14B8A6）
  - 用于：高亮、连接、强调点
- **填充**：非常微妙，如果有的话（10-20% 不透明度）

### 排版
- **字体风格**：简洁无衬线或微妙衬线
- **层级**：清晰但不戏剧性
- **间距**：充足，从不拥挤
- **处理**：纯文本，偶尔粗体强调

### 构图
- **布局**：有组织、结构化，通常网格对齐
- **留白**：丰富，呼吸空间至关重要
- **对齐**：有意图，通常居中或平衡
- **深度**：主要是 2D，通过线条重量实现微妙分层

## 最适合

- 知识可视化
- 概念映射
- 笔记式插图
- 教育内容
- 系统图表
- 思考过程
- 极简封面
- 专业/智力主题

## 避免用于

- 高能量宣传内容
- 儿童内容（过于成熟）
- 娱乐/流行文化（过于严肃）
- 任何需要鲜艳色彩的内容

## 场景适配

### 文章封面（Obsidian 风格）
```
视觉处理：
- 中心概念：单一象征性图标或对象
- 线条艺术：简洁黑色轮廓，最少阴影
- 背景：纯白或浅灰
- 标题：简洁无衬线，自然融入
- 强调：一种强调色用于关键元素
- 情绪：深思、平静、智力

示例：关于"网络效应"的文章
- 主视觉：手绘网络图节点
- 节点：黑色轮廓圆圈
- 连接：显示关系的简洁线条
- 一个节点以紫色高亮（强调）
- 标题："网络效应"在左上角
- 背景：米白色
```

### 流程图（Obsidian 风格）
```
视觉处理：
- 节点：简洁矩形和菱形
- 连接器：简单黑色箭头
- 标签：节点内的清晰无衬线文本
- 决策点：带轻微强调色的菱形
- 背景：带微妙点阵网格的浅色
- 流向：从上到下或从左到右，方向清晰

示例：用户认证流程
- 开始/结束：圆角矩形
- 流程：标准矩形
- 决策：带紫色高亮的菱形
- 箭头：简单、清晰、方向性
- 网格：用于对齐感的淡点
```

### 海报（Obsidian 风格）
```
视觉处理：
- 标题：大型、简洁排版
- 视觉：单一象征性插图
- 布局：有组织、网格对齐
- 颜色：主要单色 + 一种强调
- 层级：通过大小和间距清晰
- 背景：浅色、极简

示例：工作坊海报
- 标题："知识管理工作坊"
- 视觉：风格化大脑-网络混合图标
- 日期/时间：下方简洁文本
- CTA：带强调色的微妙高亮框
- 整体：专业、邀请但不浮夸
```

## 提示词关键词

生成 obsidian 风格图像时使用这些关键词：

**必需**：
- "手绘线条艺术"
- "简洁黑色线条"
- "白色或浅灰背景"
- "极简主义"
- "知识可视化"

**线条质感**：
- "一致线条重量"
- "平滑曲线"
- "精准但人性化"
- "非照片写实"

**颜色方向**：
- "最少使用颜色"
- "单色配单一强调"
- "微妙紫色/蓝色/青色强调"（选一种）
- "无渐变"

**构图**：
- "有组织的布局"
- "充足留白"
- "简洁构图"
- "智力美学"

**避免**：
- "五彩缤纷"
- "鲜艳"
- "详细阴影"
- "纹理背景"
- "照片写实"

## 示例提示词

### "构建第二大脑"文章封面
```
风格：Obsidian

为关于构建第二大脑的文章封面创建极简手绘插图。

视觉概念：
- 中心图像：由互连节点和线条组成的风格化大脑
- 节点：由简洁线条连接的小圆圈
- 处理：白色背景上的黑色线条艺术
- 一个节点以柔和紫色高亮（#7C3AED）
- 非常极简，智力感

标题："构建第二大脑"
- 位置：下三分之一
- 字体：简洁无衬线
- 颜色：深灰

构图：
- 居中的大脑插图
- 所有边缘周围充足留白
- 平衡、有组织
- 无额外装饰元素

线条质感：
- 手绘但精准
- 一致中等重量
- 连接上的平滑曲线
- 节点的几何圆圈

背景：纯白
情绪：深思、有组织、知识专注
```

### "内容创作流程"流程图
```
风格：Obsidian

创建显示内容创作工作流的手绘流程图。

流程：
想法 → 研究 → 大纲 → 草稿 → 编辑 → 发布

视觉处理：
- 节点：带圆角的简洁矩形
- 文本：节点内，简洁无衬线
- 连接器：节点间的简单箭头
- 背景：带微妙点阵网格的浅灰（#F5F5F5）
- 强调：以青色高亮"发布"节点（#14B8A6）

线条质感：
- 黑色线条（#2D2D2D）
- 一致中等重量
- 平滑、有意图

布局：
- 水平流，从左到右
- 间距均匀的节点
- 对齐到隐形网格
- 充足间距

情绪：清晰、有组织、专业
```

### 极简工作坊海报
```
风格：Obsidian

使用 obsidian 美学为"视觉思维工作坊"创建海报。

布局：
- 顶部：大型简洁标题"视觉思维工作坊"
- 中心：简单图标 - 单连续线绘制的灯泡
- 底部：日期/时间/地点详情

视觉：
- 灯泡：黑色线条艺术，手绘质感
- 灯丝：以紫色高亮（#7C3AED）
- 所有其他元素：黑色或深灰

背景：白色

排版：
- 标题：粗体无衬线，大号
- 详情：常规重量，较小，有组织

构图：
- 居中对齐
- 充足边距
- 所有元素周围的呼吸空间
- 无装饰额外内容

情绪：专业、邀请、智力
```

## 技术说明

### 颜色代码参考

**线条**：
- 主要：#1A1A1A（接近黑）
- 次要：#2D2D2D（深灰）
- 三级：#4A4A4A（中灰）

**背景**：
- 主要：#FFFFFF（纯白）
- 次要：#F5F5F5（米白）
- 三级：#EEEEEE（浅灰）

**强调**（每张图选一种）：
- 紫色：#7C3AED
- 蓝色：#3B82F6
- 青色：#14B8A6
- 绿色：#10B981

### 线条重量指南

- 主要元素：相当于 2-3px
- 次要细节：1-2px
- 连接：1.5px
- 高亮：3-4px 用于强调

### 间距

- 边距：画布大小的 10-15%
- 元素间距：主要组件间 5-8%
- 文本填充：舒适，从不紧凑
- 留白：越多越好

## 与其他风格对比

**vs. Notion**：更手绘，更少"数字完美"
**vs. Minimal**：更结构化，特定于知识工作
**vs. Blueprint**：更少技术性，更有机线条
**vs. Chalkboard**：更少有趣，更成熟

## 演变和变体

在保持核心 obsidian 美学的同时，您可以调整：

**更温暖的 Obsidian**：
- 背景：奶油色（#FAF9F6）
- 线条：温暖灰色（#3D3D3D）
- 强调：温暖色（橙色、琥珀色）

**高对比 Obsidian**：
- 纯黑线条（#000000）
- 纯白背景
- 无灰度

**网格重 Obsidian**：
- 可见点阵网格背景
- 更几何对齐
- 工程笔记本感觉
